# Project Epizo 🚀
### Offline-first infrastructure for a world without internet

> Building offline AI, decentralized knowledge, and resilient systems.

---

![Epizo](./assets/epizo-banner.png)

---

## 🧩 What is Epizo?

Epizo is an **offline-first personal server** that works without internet — designed for resilience, privacy, and real-world impact.

- 📚 Knowledge without internet  
- 🤖 Private AI (runs locally)  
- 🗺️ Offline navigation  
- 🎓 Education anywhere  
- 🔒 Privacy-first architecture  

---

## 🌍 Real-world use cases

**When the internet fails, Epizo continues.**

- ⚔️ Conflict zones → medical data, evacuation maps, AI triage  
- 🌪 Natural disasters → offline navigation, survival guides, coordination  
- 🏡 Remote communities → education, health info, local AI  
- 🚫 Government shutdowns → censorship-free info, mesh networking  

---

## ⚙️ Tech Stack

- **Backend:** Node.js / AdonisJS  
- **Frontend:** React / Inertia  
- **AI:** Local LLM (Ollama)  
- **Infra:** Docker, self-hosted  
- **Architecture:** Offline-first · P2P · ETH protocol layer

## 🚀 Vision

We believe access to knowledge should **not depend on connectivity**.

Epizo is building a world where:
- Internet is optional  
- AI runs locally  
- Data stays with the user  
- Systems remain operational under failure
