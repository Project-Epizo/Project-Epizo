# Project Epizo 🚀
Offline-first infrastructure for a world without internet.

Building offline AI, decentralized knowledge, and resilient systems.

## 🧩 What is Epizo?

Epizo is an offline-first personal server that provides:
- 📚 Knowledge without internet
- 🤖 Private AI (local models)
- 🗺️ Offline navigation
- 🎓 Education anywhere
- 🔒 Privacy-first architecture

## 🌍 Real-world use cases

- Conflict zones → medical + evacuation + AI triage
- Natural disasters → offline maps + survival guides
- Remote communities → education + health info
- Gov shutdowns → censorship-free + mesh networking

## ⚙️ Stack

- Backend: Node.js / AdonisJS
- Frontend: React / Inertia
- AI: Local LLM (Ollama)
- Infra: Docker, self-hosted
- Vision: Offline-first + P2P + ETH layer
